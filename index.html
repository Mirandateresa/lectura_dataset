Acerca del Proyecto

Este proyecto permite visualizar y explorar los notebooks del dataset ISCX NSL-KDD, utilizado para la detección de intrusiones en redes. Haz clic en cualquiera de las tarjetas para ver el contenido detallado de cada notebook.

Sobre el Dataset

  • Nombre: ISCX NSL-KDD
  • Propósito: Detección de intrusiones en redes
  • Mejoras: Versión mejorada del KDD'99 sin registros redundantes
  • Registros: ~125,000 en conjunto de entrenamiento
  • Características: 41 atributos + etiqueta de clase
  • Ataques: 4 categorías principales (DoS, Probe, R2L, U2R)
